Project Name: Jason's Salon

Overview:A sleek, modern landing page built for a premium salon to showcase services, pricing, and stylist expertise while providing a seamless user experience for potential clients.

Why I Built This:

I created this project to practice structuring and styling a real-world business website. My focus was on creating a highly polished, professional layout using semantic HTML for accessibility and clean CSS to match a modern, upscale brand identity.

Key Features:

Service & Pricing Menu: A beautifully organized layout making it effortless for clients to browse haircuts, styling, and color treatments.

Mobile-Responsive Design: A fluid layout built from the ground up to look perfect whether a client is booking from their phone or a desktop.

Call-to-Action Integration: Strategically placed buttons designed to drive customer engagement and maximize appointment bookings.

Tech Used:
Semantic HTML5
Custom CSS3 (Flexbox & CSS Grid Layouts)
Responsive Web Design (Media Queries)
